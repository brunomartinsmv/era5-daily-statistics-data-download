# Examples

Runnable examples for the public ERA5 daily-statistics API.

## Files

- `daily_statistics_examples.py`: examples that call
  `download_era5_daily_stats` from the main `download_era5_daily.py` module.

## Usage

Run from the repository root:

```bash
python3 examples/daily_statistics_examples.py
```

The script prints setup instructions by default. Uncomment one example function
inside the file to execute a real CDS download.
